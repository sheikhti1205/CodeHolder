# Tutorials

This folder keeps DBMS tutorial work grouped by assignment/topic.

## Folder Map

- `ERD/` - ERD project materials, task SQL, scripts, data, sample queries, and sync build files.
- `Tutorial_2/` - Tutorial 2 report, task SQL, Python scripts, assets, sample workbook, and task-wise sample queries.
- `Tutorial_3/` - Tutorial 3 report, museum data, SQL session file, Python script, ERD assets, sample queries, and build artifacts.
- `Tutorial_4/` - Task 3 workspace with prompt, references, source data, submission files, ERD outputs, and generated build files.

## Common Layout

- `Assets/` - Images and diagrams used by reports.
- `Data/` or `source_data/` - CSV/XLSX source files.
- `Report/` - Main LaTeX and PDF report outputs.
- `Sample_Query/` - Query answer files, grouped by task where needed.
- `Scripts/` - Python or helper scripts.
- `SQL/` - Schema and database task files.
- `Build_Files/` - Generated LaTeX/cache/sync files that are useful to keep but should not crowd the working files.
