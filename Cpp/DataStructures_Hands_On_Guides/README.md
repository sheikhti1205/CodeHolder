# C++ Data Structures Hands-On Workbook Guides

This folder contains beginner-friendly C++17 workbook notes for the data-structure and algorithm topics from the CSE 311/312 lecture PDFs. Each topic has its own Markdown file with concept, easy runnable code, breakdown, dry run, complexity, mistakes, and practice problems.

## How To Use These Guides

1. Read the core idea first.
2. Trace the visual model by hand.
3. Run the C++ code.
4. Change the sample input and predict the output.
5. Solve the practice problems without looking back at the code.
