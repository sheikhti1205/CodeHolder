# Official Source Files

This folder stores the main official downloadable source files that were missing from `dbms-main`.

## Downloaded Files

### BBS

- `BBS_Time_Series_Environmental_Database.xlsx`
  - Main official spreadsheet for climate, rainfall, humidity, rivers, lakes, salinity, erosion, protected areas, disasters, and other environmental indicators.
- `BBS_HBES_2024.pdf`
  - Household-based environmental survey with water source, sanitation, waste, fuel use, awareness, and related household environment data.

### DoE

- `DoE_Surface_and_Ground_Water_Quality_Report_2023.pdf`
  - Water quality measurements for rivers, lakes, and groundwater stations.
- `DoE_Ambient_Air_Quality_2018_2023.pdf`
  - Air quality report covering AQI and major pollutants.

### BRRI Climate Files

- `BRRI_Daily_Maximum_Temperature.xlsx`
- `BRRI_Daily_Minimum_Temperature.xlsx`
- `BRRI_Daily_Average_Humidity.xlsx`
- `BRRI_Daily_Sunshine.xlsx`
- `BRRI_Daily_Total_Rainfall.xlsx`

These are direct climate spreadsheets that can be loaded into Excel and normalized into station/date/value-style tables.

### BMD Reference Files

- `BMD_Normal_Max_Temperature.pdf`
- `BMD_Normal_Min_Temperature.pdf`
- `BMD_Normal_Rainfall.pdf`
- `BMD_Historical_Temperature.pdf`

These are useful as official validation and supporting reference documents.

## Suggested Use

Keep the existing CSV files in `dbms-main` as processed or combined datasets, and keep this folder as the official-source reference set.
